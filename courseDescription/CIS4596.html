CCN: CIS4596 Title: It Capstone Project Credit Hrs: 3.00
Description : 
This course will give the IT student the ability to utilize what he/she has learned from the IT Program and adapt it to a work environment. This will be accomplished by providing the student a senior project that includes first: project proposal, feasibility studies, identification of intellectual property, and a teamwork environment for project creation, and second; project support which includes: budgets, schedule management, communications through reports and presentations project testing, implementation and final approval.
