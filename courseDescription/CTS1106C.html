CCN: CTS1106C Title: Unix Credit Hrs: 3.00
Description : 
The UNIX Operating System Essentials course provides instruction in the key features and capabilities of the UNIX OS.Topics inclu e file and directory management,controlling the user work environment, archiving files and using remote commands.In addition,this course explains fundamental command-line features of the UNIX OS,including file system navigation,the vi text editor, file permissions, access control lists (ACLs), command shells, file compression, basic network use, and reading shell scripts. This course prepares students to take the Sun Certified Solaris Associate (SCSAS) Exam.
